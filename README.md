# Server Nexus (WIP)

This is a full-stack website for people that need information about Minecraft servers such as the live status, player count, required version, metadata, and more.
Users can create an account to fetch the data of any server, browse through popular servers, save their favorite servers, leave ratings/reviews on servers, and view user profiles.

## Tech Stack:

| **Front-End** | **Back-End** | **Deployment** | **APIs** |
| :--------: | :--------: | :--------: | :--------: |
| Svelte, TypeScript, <br> Tailwind CSS, <br> Daisy UI, Motion | Supabase | Vercel | MCStatus API, EmailJS |
